Небольшой, но аккуратный API для задач: регистрация, JWT, роли и CRUD.

Открыть Swagger Открыть ReDoc

  • Роли и permissions

Как начать

  1. Установите зависимости
  2. Запустите uvicorn main:app --reload
  3. Откройте Swagger

Структура

Модули разделены по зонам ответственности: база данных, JWT, схемы, permissions и endpoints.
